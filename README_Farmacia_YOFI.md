
# 💊 Farmacia YOFI Inventario

Este es un sistema web de inventario farmacéutico, 100% funcional en HTML, sin necesidad de instalar nada.

## 📁 Archivos incluidos

- `Farmacia_YOFI_Sistema_Completo_100FUNCIONAL.html` → Archivo principal del sistema.

## 🚀 ¿Cómo publicarlo como sitio web en GitHub Pages?

1. Ingresa a [https://github.com](https://github.com) y entra a tu cuenta.
2. Crea un nuevo repositorio. Ejemplo: `farmacia-yofi`
3. Haz clic en "Add file" → "Upload files"
4. Sube el archivo HTML que descargaste.
5. Haz clic en "Commit changes" para confirmar.

## 🌐 Activar GitHub Pages

1. Ve a la pestaña **Settings** del repositorio.
2. En el menú izquierdo busca **Pages** o **Pages & deployment**.
3. En la sección "Source", selecciona:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Guarda los cambios.

💡 Luego de unos segundos GitHub te mostrará un link como este:
```
https://TUUSUARIO.github.io/farmacia-yofi
```

Este será el enlace donde podrás acceder a tu sistema desde cualquier dispositivo.

## 🛡️ Importante

Este sistema guarda los datos en el navegador con LocalStorage, y funciona sin conexión a internet. Si deseas hacer respaldos, puedes usar versiones futuras con exportación/importación JSON.
